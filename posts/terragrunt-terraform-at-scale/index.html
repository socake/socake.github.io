Terragrunt еӯҳеңЁзҡ„зҗҶз”ұ
#

TerraformпјҲжҲ– OpenTofuпјүз”ЁеҲ°еҗҺйқўйғҪдјҡзў°еҲ°еҗҢдёҖдёӘй—®йўҳпјҡstate жӢҶеҲҶд»ҘеҗҺпјҢз®ЎзҗҶеӨҡдёӘ state еҸҳеҫ—йқһеёёз—ӣиӢҰгҖӮ

дёҖдёӘе…ёеһӢзҡ„ “еҸӘз”Ё Terraform” йЎ№зӣ®й•ҝиҝҷж ·пјҡ

terraform/
в”ңв”Җв”Җ main.tf      # 500 иЎҢ all-in-one
в”ңв”Җв”Җ variables.tf
в”ңв”Җв”Җ outputs.tf
в””в”Җв”Җ backend.tf

еҚ•дёӘ stateпјҢжүҖжңүиө„жәҗйғҪеңЁдёҖиө·гҖӮеҲҡејҖе§ӢеҫҲзҲҪпјҢдҪҶеҫҲеҝ«еҮәзҺ°й—®йўҳпјҡ

  • terraform plan иҰҒ 2 еҲҶй’ҹпјҲиө„жәҗи¶ҠжқҘи¶ҠеӨҡпјү
  • ж”№дёҖдёӘ VPC й…ҚзҪ®иҰҒ plan ж•ҙдёӘз”ҹдә§зҺҜеўғ
  • blast radius е·ЁеӨ§пјҡе°Ҹй”ҷиҜҜеҸҜиғҪиҜҜеҲ  RDS
  • дёҖдёӘж–Үд»¶ 5000 иЎҢжІЎжі•зңӢ

жүҖжңүдәәзҡ„з¬¬дёҖеҸҚеә”йғҪжҳҜ жӢҶ stateгҖӮжҢүжЁЎеқ—гҖҒжҢүзҺҜеўғгҖҒжҢүиҙҰеҸ·жӢҶпјҡ

terraform/
в”ңв”Җв”Җ modules/
в”Ӯ   в”ңв”Җв”Җ vpc/
в”Ӯ   в”ңв”Җв”Җ eks/
в”Ӯ   в””в”Җв”Җ rds/
в”ңв”Җв”Җ envs/
в”Ӯ   в”ңв”Җв”Җ dev/
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ vpc/
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ main.tf
в”Ӯ   в”Ӯ   в”Ӯ   в”ңв”Җв”Җ backend.tf
в”Ӯ   в”Ӯ   в”Ӯ   в””в”Җв”Җ ...
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ eks/
в”Ӯ   в”Ӯ   в””в”Җв”Җ rds/
в”Ӯ   в”ңв”Җв”Җ staging/
в”Ӯ   в”Ӯ   в””в”Җв”Җ ... (дёҖжЁЎдёҖж ·зҡ„з»“жһ„)
в”Ӯ   в””в”Җв”Җ prod/
в”Ӯ       в””в”Җв”Җ ... (еҸҲдёҖж¬ЎеӨҚеҲ¶)

зҺ°еңЁж–°й—®йўҳеҮәжқҘдәҶпјҡ

  1. backend.tf еҲ°еӨ„еӨҚеҲ¶пјҡжҜҸдёӘзӣ®еҪ•йғҪиҰҒеҶҷ bucketгҖҒkeyгҖҒregionгҖҒdynamodb_tableгҖӮ30 дёӘзӣ®еҪ•е°ұжҳҜ 30 д»ҪгҖӮ
  2. provider й…ҚзҪ®еҲ°еӨ„еӨҚеҲ¶пјҡregionгҖҒassume_roleгҖҒdefault_tagsгҖӮеҸҲжҳҜ 30 д»ҪгҖӮ
  3. еҸҳйҮҸз©ҝйҖҸйә»зғҰпјҡdev/vpc/outputs йҮҢзҡ„ vpc_id иҰҒз»ҷ dev/eks/main.tf з”ЁпјҢеҸӘиғҪз”Ё data.terraform_remote_state жүӢеҠЁеҶҷдёҖйҒҚгҖӮ
  4. и·Ё state йЎәеәҸпјҡvpc иҰҒе…ҲдәҺ eks applyгҖӮжІЎжңүе·Ҙе…·дҝқиҜҒйЎәеәҸпјҢйқ дәәи®°еҝҶгҖӮ
  5. жү№йҮҸж“ҚдҪңпјҡз”ҹдә§еҚҮзә§ Kubernetes зүҲжң¬пјҢиҰҒ apply 20 дёӘ stateпјҢжүӢеҠЁ cd + terraform apply ж•ІеҲ°еҗҗгҖӮ

Terragrunt жҳҜиҝҷдәӣй—®йўҳзҡ„зі»з»ҹи§Јжі•гҖӮе®ғжҳҜ Terraform/OpenTofu зҡ„ wrapperпјҢз”ЁдёҖд»Ҫ terragrunt.hcl е®ҡд№ү “жҖҺд№Ҳи°ғ Terraform”пјҢиҖҢдёҚжҳҜ"иҰҒйғЁзҪІд»Җд№Ҳ"гҖӮTerraform з»§з»ӯз®ЎеҹәзЎҖи®ҫж–ҪжҸҸиҝ°пјҢTerragrunt з®ЎеҹәзЎҖи®ҫж–Ҫзј–жҺ’гҖӮ

Terragrunt зҡ„ж ёеҝғжҠҪиұЎ
#

дёҖдёӘжңҖе°Ҹзҡ„ terragrunt.hcl
#

live/
в”ңв”Җв”Җ terragrunt.hcl         # ж №й…ҚзҪ®пјҡbackend, provider
в””в”Җв”Җ prod/
    в””в”Җв”Җ us-west-2/
        в””в”Җв”Җ vpc/
            в””в”Җв”Җ terragrunt.hcl   # еҚ•е…ғй…ҚзҪ®пјҡinputs

ж №й…ҚзҪ® live/terragrunt.hclпјҡ

# дёәжүҖжңүеӯҗеҚ•е…ғз”ҹжҲҗ backend
remote_state {
  backend = "s3"
  generate = {
    path      = "backend.tf"
    if_exists = "overwrite"
  }
  config = {
    bucket         = "my-company-tfstate"

    region         = "us-west-2"
    encrypt        = true
    dynamodb_table = "tf-locks"
  }
}

# дёәжүҖжңүеӯҗеҚ•е…ғз”ҹжҲҗ provider
generate "provider" {
  path      = "provider.tf"

  contents  = <<EOF
provider "aws" {
  region = "us-west-2"
  default_tags {
    tags = {
      ManagedBy   = "terragrunt"
      Environment = "prod"
    }
  }
}
EOF
}

# е…ЁеұҖеҸҳйҮҸ
inputs = {
  environment = "prod"

}

Terragrunt иғҢеҗҺеҒҡзҡ„дәӢпјҡ

  1. и§Јжһҗ includeпјҡеҠ иҪҪзҲ¶зӣ®еҪ•зҡ„ terragrunt.hclпјҢеҗҲе№¶ remote_stateгҖҒgenerateгҖҒinputs
  2. з”ҹжҲҗ backend.tfгҖҒprovider.tfпјҡеҶҷеҲ°дёҙж—¶зӣ®еҪ•пјҲ.terragrunt-cache/пјү
  3. дёӢиҪҪ sourceпјҡеҰӮжһң source жҳҜ gitпјҢе…ӢйҡҶеҲ°дёҙж—¶зӣ®еҪ•
  4. и°ғз”Ё Terraformпјҡterraform init з”Ёз”ҹжҲҗзҡ„ backendпјҢterraform plan дј е…Ҙ inputs

иҝҷж ·жҜҸдёӘеӯҗеҚ•е…ғзҡ„ terragrunt.hcl еҸӘеҶҷ"иҮӘе·ұзӢ¬жңүзҡ„й…ҚзҪ®"пјҢbackend/provider е…ЁйғЁз»§жүҝиҮӘж №гҖӮ100 дёӘ state зҡ„йЎ№зӣ®пјҢbackend.tf еҸӘеҶҷдёҖж¬ЎгҖӮ

find_in_parent_folders е’Ң include
#

find_in_parent_folders() дјҡд»ҺеҪ“еүҚзӣ®еҪ•еҫҖдёҠжүҫжңҖиҝ‘зҡ„ terragrunt.hclгҖӮе®ғжҳҜ Terragrunt DRY жЁЎејҸзҡ„ж ёеҝғгҖӮ

еӨҡзә§ includeпјҡ

live/
в”ңв”Җв”Җ terragrunt.hcl              # е…ЁеұҖпјҲbackendпјү
в”ңв”Җв”Җ _env/
в”Ӯ   в””в”Җв”Җ prod.hcl                # prod зҺҜеўғзү№жңү
в””в”Җв”Җ prod/
    в””в”Җв”Җ us-west-2/
        в””в”Җв”Җ _region/
            в””в”Җв”Җ us-west-2.hcl   # region зү№жңү
        в””в”Җв”Җ vpc/
            в””в”Җв”Җ terragrunt.hcl

жҜҸж¬Ўеј•з”ЁйғҪиҰҒеҶҷдёҖйҒҚ data blockпјҢеҮәй”ҷзҺҮй«ҳгҖӮ

Terragrunt зҡ„ dependency block жҳҜдјҳйӣ…зҡ„жӣҝд»Јпјҡ

# live/prod/us-west-2/eks/terragrunt.hcl
include "root" {
  path = find_in_parent_folders()
}

dependency "vpc" {
  config_path = "../vpc"

  mock_outputs = {
    vpc_id             = "vpc-mock"
    private_subnet_ids = ["subnet-mock-1", "subnet-mock-2"]
  }
  mock_outputs_allowed_terraform_commands = ["validate", "plan"]
}

terraform {
  source = "git::...//modules/eks?ref=v1.5.0"
}

inputs = {
  vpc_id     = dependency.vpc.outputs.vpc_id
  subnet_ids = dependency.vpc.outputs.private_subnet_ids
  cluster_name = "prod-main"
}

Terragrunt еңЁ apply eks д№ӢеүҚдјҡпјҡ

  1. жЈҖжҹҘ ../vpc жҳҜеҗҰе·Із»Ҹ apply иҝҮ
  2. д»Һ ../vpc зҡ„ state йҮҢиҜ» outputs
  3. жҠҠ outputs жіЁе…ҘеҲ°еҪ“еүҚ inputs

mock_outputs зҡ„дҪңз”Ёпјҡ../vpc иҝҳжІЎ apply ж—¶пјҢplan/validate йҳ¶ж®өз”Ё mock еҖјд»ЈжӣҝпјҢи®©дҪ иғҪеңЁ dev зҺҜеўғзңӢеҲ° plan иҖҢдёҚжҳҜжҠҘй”ҷгҖӮ

dependencies blockпјҡеҸӘе®ҡд№үжү§иЎҢйЎәеәҸ
#

dependencies blockпјҲеӨҚж•°пјүеҸӘеЈ°жҳҺйЎәеәҸпјҢдёҚиҜ» outputпјҡ

dependencies {
  paths = ["../vpc", "../iam"]
}

еёёз”ЁдәҺ “A еҝ…йЎ»еңЁ B д№ӢеүҚ apply” дҪҶ A дёҚйңҖиҰҒиҜ» B зҡ„ outputгҖӮе’Ң dependency (еҚ•ж•°) зҡ„еҢәеҲ«пјҡ

зү№жҖ§dependencydependencies
иҜ»еҸ– outputsжҳҜеҗҰ
еҪұе“Қжү§иЎҢйЎәеәҸжҳҜжҳҜ
ж”ҜжҢҒ mockжҳҜеҗҰ
й…ҚзҪ®ж–№ејҸдёҖдёӘ block дёҖдёӘдҫқиө–дёҖдёӘ block еӨҡдёӘи·Ҝеҫ„

run-allпјҡжү№йҮҸж“ҚдҪңзҡ„е‘Ҫд»Ө
#

зңҹжӯЈзҡ„и§„жЁЎеҢ–иҰҒйқ  run-allгҖӮ

# еңЁж•ҙдёӘ live/prod зӣ®еҪ•дёӢжҢүдҫқиө–йЎәеәҸ plan жүҖжңү state
cd live/prod
terragrunt run-all plan

# apply жүҖжңү
terragrunt run-all apply

# еҸӘзңӢжҹҗдёҖз»„зҡ„ graph
terragrunt graph-dependencies

run-all еҒҡзҡ„дәӢпјҡ

  1. йҖ’еҪ’жү«жҸҸеҪ“еүҚзӣ®еҪ•дёӢжүҖжңү terragrunt.hcl
  2. и§ЈжһҗжҜҸдёӘ unit зҡ„ dependency / dependencies block жһ„е»ә DAG
  3. жҢү DAG жӢ“жү‘йЎәеәҸи°ғз”Ё TerraformпјҲж— дҫқиө–зҡ„е№¶еҸ‘пјү
  4. жҜҸдёӘ unit зҡ„ stdout иҒҡеҗҲеҲ°дёҖиө·иҫ“еҮә

run-all зҡ„з”ҹдә§еқ‘
#

еқ‘ 1пјҡе№¶еҸ‘ж•°иҝҮй«ҳжү“зҲҶ API rate limit

жӣҙжҲҗзҶҹзҡ„йҖүжӢ©жҳҜ SpaceliftпјҢе®ғеҺҹз”ҹж”ҜжҢҒ TerragruntпјҢжңүеҸҜи§ҶеҢ– UIгҖҒdrift detectionгҖҒpolicy йӣҶжҲҗгҖӮ

Drift Detection
#

Terragrunt жң¬иә«дёҚеҒҡ drift detectionпјҢйңҖиҰҒйўқеӨ–е·Ҙе…·гҖӮжҲ‘д»¬зҡ„еҒҡжі•пјҡ

  • жҜҸеӨ©еҮҢжҷЁе®ҡж—¶и·‘ run-all planпјҢplan з»“жһңе’ҢдёҠж¬ЎжҜ”иҫғ
  • е·®ејӮеҸ‘й’үй’үпјҡеҰӮжһңжңүиө„жәҗжјӮз§»пјҢalert з»ҷиҝҗз»ҙ
  • жјӮз§»еёёи§ҒеҺҹеӣ пјҡжүӢеҠЁеңЁ console ж”№гҖҒйқһ Terragrunt з®ЎзҗҶзҡ„е·Ҙе…·пјҲжҜ”еҰӮ ASG иҮӘеҠЁи°ғе®№йҮҸпјү

иҗҪең°иё©еқ‘
#

еқ‘ 1пјҡ.terragrunt-cache еҚ зӣҳ
#

Terragrunt жҜҸж¬Ў run дјҡеңЁжҜҸдёӘ unit дёӢеҲӣе»ә .terragrunt-cache/ зӣ®еҪ•пјҢдёӢиҪҪ sourceгҖҒдҝқеӯҳ planгҖӮдёҖдёӘеӨ§еһӢйЎ№зӣ®иҝҷдёӘзӣ®еҪ•еҸҜиғҪеҚ  10+ GBгҖӮ

жё…зҗҶпјҡ

find . -type d -name .terragrunt-cache -exec rm -rf {} +

CI йҮҢжҜҸж¬ЎиҝҗиЎҢеҗҺдё»еҠЁжё…зҗҶгҖӮ

еқ‘ 2пјҡdependency mock еҶҷй”ҷеҜјиҮҙз”ҹдә§дәӢж•…
#

mock_outputs зҡ„зӣ®зҡ„жҳҜи®© plan иғҪи·‘пјҢдҪҶеҰӮжһңдҪ дёҚе°ҸеҝғеңЁ apply ж—¶д№ҹз”ЁдәҶ mockпјҲиҜҜй…Қ mock_outputs_allowed_terraform_commandsпјүпјҢз”ҹдә§дјҡеә”з”Ё mock еҖјвҖ”вҖ”жҜ”еҰӮ subnet_ids = ["subnet-mock"] е°ұзңҹеҺ»еҲӣе»әдёҖдёӘдёҚеӯҳеңЁзҡ„ subnet зҡ„ EKSпјҢзӣҙжҺҘеӨұиҙҘжҲ–жӣҙзіҹгҖӮ

и§„еҲҷпјҡmock_outputs_allowed_terraform_commands еҸӘеҲ— ["validate", "plan"]пјҢж°ёиҝңдёҚиҰҒеҠ  applyгҖӮ

еқ‘ 3пјҡrun-all зҡ„ dependency и·Ёзӣ®еҪ•й—®йўҳ
#

dependency "../vpc" еј•з”ЁзӣёеҜ№и·Ҝеҫ„гҖӮеҰӮжһңдҪ йҮҚжһ„зӣ®еҪ•з»“жһ„пјҢжүҖжңүдҫқиө–йғҪеҫ—ж”№гҖӮиҝҷжҳҜ Terragrunt 1.0 д№ӢеүҚзҡ„з—ӣзӮ№гҖӮ

Stacks жЁЎејҸдёӢдҫқиө–жҳҜйҖҡиҝҮ values дј е…Ҙ unit зҡ„пјҢж”№зӣ®еҪ•дёҚеҪұе“Қдҫқиө–иЎЁиҫҫгҖӮиҝҷжҳҜ Stacks еёҰжқҘзҡ„йҡҗжҖ§еҘҪеӨ„гҖӮ

еқ‘ 4пјҡTerragrunt зүҲжң¬еҚҮзә§з ҙеқҸе…је®№
#

Terragrunt еҺҶеҸІдёҠеҒҡиҝҮеҮ ж¬Ўе°Ҹзҡ„ breaking changeпјҲжҜ”еҰӮ dependency block иҜӯд№үи°ғж•ҙпјүгҖӮз”ҹдә§дёҠй”Ғе®ҡ Terragrunt зүҲжң¬пјҢеҚҮзә§еүҚеңЁ staging и·‘е…ЁйҮҸ planпјҡ

# .terragrunt-version
v0.78.2

й…ҚеҗҲ tgenv жҲ– asdf иҮӘеҠЁеҲҮжҚўзүҲжң¬гҖӮ

еқ‘ 5пјҡrun-all е№¶еҸ‘дёӢзҡ„ state lock еҶІзӘҒ
#

дёӨдёӘ unit еҗҢж—¶еҜ№еҗҢдёҖдёӘ DynamoDB lock table з«һдәүпјҢеҸҜиғҪ deadlockгҖӮеӨ§йҮҸ unit е№¶еҸ‘ж—¶еҒ¶е°”зңӢеҲ° Error locking stateгҖӮ

зј“и§ЈпјҡйҷҚдҪҺ --terragrunt-parallelismпјҢжҲ–з»ҷжҜҸдёӘзҺҜеўғзӢ¬з«Ӣзҡ„ lock tableгҖӮ

д»Җд№Ҳж—¶еҖҷдёҚз”Ё Terragrunt
#

  • State е°‘дәҺ 10 дёӘпјҡзӣҙжҺҘеҶҷ Terraform жӣҙз®ҖеҚ•
  • еӣўйҳҹдёҚзҶҹ HCLпјҡеӯҰжӣІзәҝеҸ еҠ пјҢеј•е…Ҙж–°е·Ҙе…·еўһеҠ еҝғжҷәиҙҹжӢ…
  • е®Ңе…Ё Pulumi дҪ“зі»пјҡPulumi жңүиҮӘе·ұзҡ„ Stack + ComponentпјҢTerragrunt з®ЎдёҚдәҶ

Terragrunt зҡ„з”ңиңңеҢәжҳҜпјҡ“е·Із»Ҹж·ұеәҰз”Ё Terraform/OpenTofu + state и¶…иҝҮ 20 дёӘ + жңүеӨҡзҺҜеўғ/еӨҡ region з®ЎзҗҶйңҖжұӮ”гҖӮе°ҸдәҺиҝҷдёӘи§„жЁЎзәҜ Terraform е°ұеӨҹгҖӮ

2023 е№ҙд№ӢеҗҺ IaC дё–з•ҢеҸҳдәҶпјҡHashiCorp жҠҠ Terraform ж”№жҲҗ BSLпјҢLinux Foundation жҺҘз®ЎдәҶ OpenTofuгҖӮPulumi дҫқз„¶еңЁд»Јз ҒејҸ IaC зҡ„и·ҜдёҠеқҡжҢҒгҖӮеӣўйҳҹйҖүеһӢж—¶йқўеҜ№зҡ„дёҚжҳҜ Terraform дёҖе®¶зӢ¬еӨ§пјҢиҖҢжҳҜдёүжқЎжҠҖжңҜи·Ҝзәҝзҡ„зңҹе®һеҜ№жҜ”гҖӮжң¬ж–ҮиҜ•еӣҫз»ҷеҮәдёҖдёӘдёҚеҒҸдёҚеҖҡзҡ„зӯ”жЎҲгҖӮ

OpenTofu е®һжҲҳпјҡејҖжәҗ Terraform з®ЎзҗҶ AWS е’ҢйҳҝйҮҢдә‘еҹәзЎҖи®ҫж–Ҫ

2025-06-18·1106 еӯ—·6 еҲҶй’ҹ
